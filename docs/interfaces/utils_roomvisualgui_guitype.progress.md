# Interface: Progress

[utils/roomVisualGUI/GUIType](../modules/utils_roomvisualgui_guitype.md).Progress

可用属性：
width：进度条宽度
height:进度条高度
value:进度值0~100
background:进度条背景颜色
progressColor:进度条颜色
borderColor:进度条边框颜色
visibility:是否可见
